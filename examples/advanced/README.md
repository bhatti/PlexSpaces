# Advanced Examples

This directory contains advanced examples demonstrating complex distributed systems patterns and algorithms.

## Examples

### Byzantine Generals

**Location**: `byzantine/`

Demonstrates Byzantine fault tolerance with consensus algorithms.

**Features**:
- Byzantine fault tolerance
- Consensus algorithms
- Fault injection
- Multi-node coordination

**See**: [Byzantine Generals README](byzantine/README.md)

### N-Body Simulation

**Location**: `nbody/`

Complex physics simulation with multi-actor coordination.

**Features**:
- Physics simulation
- Multi-actor coordination
- Performance optimization

**See**: [N-Body Simulation README](nbody/README.md)

### N-Body WASM

**Location**: `nbody-wasm/`

WASM-based simulation with TypeScript actors.

**Features**:
- WASM-based simulation
- TypeScript actors
- Performance optimization

**See**: [N-Body WASM README](nbody-wasm/README.md)
